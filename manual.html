Manual del Usuario de Tonayu

Guía visual y educativa para aprender a programar en náhuatl

1. Descripción del Sistema

Tonayu es un entorno interactivo web diseñado para enseñar programación utilizando un lenguaje estructurado basado en el náhuatl...

2. Requisitos de Uso

  • Navegador web moderno (Chrome, Firefox, Edge, Safari)
  • Computadora o dispositivo móvil con Windows, Linux o macOS
  • No requiere conexión a internet
  • No se requiere experiencia previa en programación

3. Guía Paso a Paso para Usar la Interfaz

Área de Escritura (Tlàhtoltecpantiliztli): Aquí escribes el código usando sintaxis náhuatl. 

chaj x = 10;
chaj y = 2;
chaj z = (x + y) + 80;
qex z;
qex x + y;
qex x - y;
qex x * y;
qex x / y;
qex x % y;

Botón "Ticitaliz": Ejecuta el análisis del código en tiempo real.

Panel de análisis: Muestra errores o confirma que todo funciona bien.

Tabla de Tokens (Tlahtoltlàmàchilistli): Lista los elementos reconocidos, tipo, valor y traducción.

4. Errores Comunes y Cómo Solucionarlos

Error mostrado Causa probable Solución
Esperaba 'b’ah' Falta el marcador de inicio de bloque Agrega 'b’ah' antes de las instrucciones
Símbolo inesperado Error ortográfico o caracter mal escrito Corrige el símbolo
Identificador no declarado Variable usada sin declararse Agrega la instrucción 'chaj'
Falta ';' Instrucción incompleta Coloca punto y coma al final

5. Idioma

La interfaz está disponible en náhuatl y español. Puedes cambiar el idioma desde el menú superior.

6. Glosario de Comandos Principales

  • chaj – Declarar variable
  • toj – Condición "si"
  • b’ah – Inicio de bloque
  • nik’ – Fin de bloque
  • qex – Imprimir
  • utz' – Retornar valor

7. Propósito Educativo

Tonayu es una herramienta educativa inclusiva, diseñada para enseñar lógica, estructuras de programación y resolución de problemas usando una lengua originaria. Es ideal para clases, talleres, autoaprendizaje y fortalecimiento cultural.

8. Características del Equipo

Requisito Especificación
Dispositivo Computadora o móvil
Navegador web Chrome, Firefox, Edge, Safari
Procesador Mínimo doble núcleo
Memoria RAM Al menos 2 GB
Sistema operativo Compatible con UTF-8

9. Ejemplos y Casos de Uso

  • Clases comunitarias: Enseñar lógica básica en náhuatl
  • Autoaprendizaje desde celulares: Practicar sin experiencia previa
  • Talleres de alfabetización digital: Para adultos mayores
  • Proyectos escolares: Automatizar tareas simples o cálculos

10. ¿Por Qué Vale la Pena Tonayu?

Tonayu rompe la barrera del idioma en la programación. Permite aprender de forma más significativa, natural y culturalmente conectada. Es accesible, portátil y útil para preservar la lengua náhuatl en el contexto tecnológico.
